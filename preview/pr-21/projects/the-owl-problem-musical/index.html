The Owl Problem Musical

Event: Agile Testing Days 2024 Date: Wednesday, November 20th, 2024, 8:30 p.m. – 9:30 p.m. Location: Room F1+F2+F3 - Plenary Type: Social Event - Musical Performance

Overview

The Owl Problem Musical is described as "the first-ever, full-blown Agile Testing Days MUSICAL" - a theatrical production that explores contemporary workplace conflicts in software testing through music, storytelling, and theatrical performance.

The Story

The narrative centers on a technology company's testing leadership implementing an advanced AI assistant, which triggers tension between automation advocates and those prioritizing human collaboration. The story escalates dramatically, culminating in a team member's disappearance.

Format

This amateur theatrical production combines:

  • Storytelling and narrative
  • Musical performances
  • Theatrical combat sequences
  • An "evil AI" antagonist
  • Professional and amateur performers from the testing community

Key Themes

  • The impact of AI and automation on software testing
  • Tension between technological advancement and human collaboration
  • Workplace dynamics and team conflicts
  • The role of human judgment in automated environments

Cast & Crew

The production features a collaborative cast of performers from the testing and development community:

  • Lena Nyström (CEO and Senior Test Consultant @ TestScouts Stockholm AB)
  • Samuel Nitsche (Senior Software Developer @ QualityMinds GmbH)
  • Veerle Verhagen (Tester @ NS)
  • Tamara Josten (Junior Test Automation Engineer @ QualityMinds GmbH)
  • Rachel Kibler (Quality Engineer @ 1-800 Contacts)
  • Bastian Knerr (Teamlead @ QualityMinds)
  • Callum Akehurst-Ryan (Staff Quality Engineer @ Artlogic)

Availability

The performance was available for virtual pass attendees via streaming.
